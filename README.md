Customer Shopping Behavior Analysis
Overview

This project analyzes customer shopping behavior using transactional data containing 3,900+ purchase records across multiple product categories. The objective was to uncover actionable business insights related to customer demographics, spending patterns, subscription trends, discount behavior, and product performance.

The project combines Python, PostgreSQL, and Power BI to perform end-to-end data analysis and dashboard visualization.

Tech Stack
Python (Pandas, NumPy)
PostgreSQL
Power BI
Excel / CSV Dataset
Project Workflow
1. Data Cleaning & Preparation (Python)
Handled missing values
Standardized column names
Performed feature engineering
Removed redundant columns
Exported cleaned dataset to PostgreSQL
2. SQL Business Analysis

Executed analytical SQL queries to answer business-focused questions such as:

Revenue contribution by gender
High-spending discount users
Customer segmentation
Subscription behavior analysis
Top-performing products
Revenue by age group
Shipping type comparison
3. Dashboard Visualization (Power BI)

Created an interactive dashboard featuring:

KPI Cards
Revenue Analysis
Customer Segmentation
Subscription Insights
Category-wise Sales
Dynamic Filters & Slicers
Key Insights
Male customers generated nearly 2× more revenue than female customers.
Young Adults contributed the highest revenue share.
Majority of repeat buyers were still unsubscribed, indicating conversion opportunities.
Clothing category dominated both sales volume and revenue.
Express shipping users showed slightly higher average spending.
Business Recommendations
Convert repeat buyers into subscribers through targeted campaigns.
Optimize discount-heavy products to improve margins.
Increase focus on young adult customer acquisition.
Promote premium express shipping options.
Highlight top-rated products in marketing campaigns.
